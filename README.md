# Unity 2D Elevator Simulation

## Project Overview
This project is a **2D Elevator Simulation** developed using **Unity Engine** and **C#**.  
The system simulates the behavior of multiple elevators responding to floor requests in a building.

The simulation includes **three elevators and four floors**, where the **nearest available elevator responds to floor requests**. Each elevator moves smoothly between floors and maintains its own request queue.

---

## Features
- 3 independent elevators
- Minimum of 4 floors (Ground, 1, 2, 3)
- Floor call buttons for each floor
- Nearest elevator selection algorithm
- Individual request queue for each elevator
- Smooth elevator movement between floors
- Current floor display for each elevator
- Prevents multiple elevators responding to the same request

---

## Technologies Used
- **Unity Engine**
- **C#**
- **Unity UI System**
- **Coroutine-based movement**
